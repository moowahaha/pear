Moowahaha's PEAR Channel

Registering the channel: 
pear channel-discover github.com/moowahaha/pear
Listing available packages: 
pear remote-list -c moowahaha
Installing a package: 
pear install moowahaha/package_name
Installing a specific version/stability: 
pear install moowahaha/package_name-1.0.0
pear install moowahaha/package_name-beta
Receiving updates via a feed: 
http://www.github.com/moowahaha/pear/feed.xml
